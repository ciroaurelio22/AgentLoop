    How Agent Loop works

    Agent Loop keeps a task queue in your repo. Use this console to plan tasks. Run work from a coding agent in Cursor, Claude Code, or Codex.

    1. Create a task: set a title, edit the program, click Save.
    2. Finish setup and turn on autostart if the gate asks for it.
    3. Open a new agent chat in this workspace.
    4. Send a short message. Examples below.
    5. The agent loads the queue, takes the next pending task, and executes it.

    Examples

    1. Create a task in the console

    • Title: Improve mobile UX
    • Program file: specs/agent-tasks/TASK-002.md
    • Fill Objective and Acceptance (- [ ] checkboxes), then Save

    2. Messages you can send in a new agent chat

    • go
    • start
    • next task
    • vai

    Any short nudge works. You do not need to paste the full program.

    3. What the agent does

    • Reads .agent-loop/queue.json
    • Opens specs/agent-tasks/TASK-002.md
    • Implements the scope, runs lint/tests if needed, ticks acceptance items

    Run AI here when you want the agent to draft or edit the open program from the console. Verify checks acceptance boxes in the saved file.
